# KSP Flight Control System — v0.1 (Linear Thrust Ramp)

A deterministic flight test framework for Kerbal Space Program using a time-consistent linear throttle ramp, detection system, and structured logging.

## Goals
- Predictable input
- Measurable output
- Reliable detection
- Capture liftoff throttle

## Structure
- TestRun
- ThrottleRampController
- DetectionSystem
- RunLogger
- FlightController
- KRPCClient
- AnalysisTool

## Usage
Run FlightController, then analyze logs.
